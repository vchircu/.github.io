How to identify Shotgun Surgery using NDepend

In this article we’ll see how to identify an afferent (incoming) coupling code smell: Shotgun Surgery.

Shotgun SurgeryÂ Detection Strategy

A method suffers from Shotgun Surgery if it is called many times from many other classes.Â Object-Oriented Metrics in Practice, by Michele Lanza and Radu Marinescu,Â proposes the followingÂ detection strategy for Shotgun Surgery:

(CM > Short Memory Cap) AND (CC > Many)

This detection strategy uses twoÂ metrics:

  • CM – Changing Methods – the number of methods that callÂ the measured method
  • CC – Changing Classes – the number of classes in which the changing methods are defined

This detection strategy usesÂ Generally-Accepted Meaning Thresholds. I used 7 as the value for Short Memory Cap and 10 for Many.

CM – Changing Methods

This metric counts the number of distinct methods that callÂ the measured method. We only care about our own code, for both caller methods and measured method. But, since framework code doesn’t call our code, we can rely on theÂ NbMethodsCallingMe NDepend metric.

 
// &lt;Name&gt;CM&lt;/Name&gt;
from m in JustMyCode.Methods
let cm = m.NbMethodsCallingMe
orderby cm descending
select new { m, cm }

CC – Changing Classes

This metric counts the the number of classes in which the changing methods are defined

// &lt;Name&gt;CC&lt;/Name&gt;
from m in JustMyCode.Methods
let changingClasses = m.MethodsCallingMe.Select(method =&gt; method.ParentType).ToHashSet()
let cc = changingClasses.Count()
orderby cc descending
select new { m, cc, changingClasses }

Putting it all together

Now that we know how to compute each of the required metrics, letâ€™s see how the detection strategy looks like:

// &lt;Name&gt;Shotgun Surgery&lt;/Name&gt;
warnif count &gt; 0
// ** Thresholds **
let ShortMemoryCap = 7
let Many = 10

// ** Detection Strategy **
from m in JustMyCode.Methods
let methodsCallingMe = m.MethodsCallingMe
let changingClasses = methodsCallingMe.Select(method =&gt; method.ParentType).ToHashSet()
let cm = m.NbMethodsCallingMe
let cc = changingClasses.Count()

where 
 // Operation is called by too many other methods
 cm &gt; ShortMemoryCap &amp;&amp; 
 // Incoming calls are from many classes
 cc &gt; Many
select new { m, cm, cc, changingClasses, methodsCallingMe }
